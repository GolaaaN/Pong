
# Pong

A pong game created with C++ and SFML.


## Demo




## How to play?

Clone the project

```bash
  git clone https://github.com/GolaaaN/Pong
```

Go to the project directory

```bash
  cd Pong
```

Install dependencies

You have to install the SFML library and connect it to visual studio.
